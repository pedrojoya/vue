Vinculación bidireccional

Mediante la directiva 

v-model="var_modelo"
hacemos que el texto introducido por el usuario en el input será vinculado con la variable indicada del modelo de datos.

Mediante la directiva 

v-text="expr"
hacemos que el resultado de evaluar la expresión se establezca como contenido de la etiqueta

Usando 

{{expr}}
dentro del contenido de una etiqueta hacemos que se evalúe la expresión y se muestre su resultado

Por ejemplo, el texto introducido en el siguiente cuandro de texto será almacenado en una variable del modelo que será mostrado en los dos párrafos siguientes

La variable mensaje del modelo de datos vale: {{mensaje}}

Vinculación de atributos

Mediante la directiva v-bind:atributo="expr". Dicha directiva puede ser acortada usando :atributo="expr". Puede ser usada con cualquier atributo
